The Duolingo HLR full data file is hosted externally and may require manual download.

Manual step:
  1. Open the Duolingo HLR repository README.
  2. Follow its dataset link.
  3. Download the gzipped CSV.
  4. Save it as:
     data/raw/duolingo_hlr/learning_traces.csv.gz

Alternatively run this script with:
  --duolingo-raw /path/to/learning_traces.csv.gz
